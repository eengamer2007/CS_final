# running
## installing dependencies
### debian based system
```
$ /bin/bash ./install.sh
```
### other
install the dependencies listen at the [dependencies](#dependencies) section
## running the code
### with marking as executable
linux only
to mark as executable
```
$ chmod +x ./main.py
```
to run after that
```
$ ./main.py
```
### without mark as executable
use this for windows
```
$ python3 main.py
```
